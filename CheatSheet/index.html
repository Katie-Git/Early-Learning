Making HTML Tables Cheatsheet

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead Table Head The set of rows defining the calumn headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The cell data container.
<tfoot> Table Foot The set of rows containing the footer in the table.
